Welcome to Jingjing's Flask Presidents Demo

This is a full-stack web demo using Python Flask. It allows users to:

  • Upload a CSV file of U.S. Presidents
  • Import the data into a MySQL database
  • Download the CSV template file
  • View detailed information in table form
  • Explore card-style profiles of each President
  • Click to visit each President's Wikipedia entry

The project demonstrates Flask routing, form handling, file upload, template rendering with Jinja, and database operations with MySQL.
